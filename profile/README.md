## Overview

Photomator Advanced is an AI-powered photo editing application from the Pixelmator Team that uses machine learning to automate and enhance color adjustments, exposure correction, white balance, crop composition, and retouching with one-click intelligence that analyzes each photo individually. It serves photographers who want professional-quality photo enhancement without the manual adjustment expertise that traditional photo editors require. The ML engine runs entirely on-device using the Neural Engine for fast, private processing.

ML-powered auto adjustments analyze each photo's content, lighting, color, and composition to apply individually tailored corrections that a skilled photographer would make manually. The auto adjustment handles exposure, white balance, tone curve, saturation, vibrance, and sharpening with a single click, producing results calibrated to each specific image rather than applying generic corrections. Manual refinement is available for every ML-suggested adjustment.

Color adjustments provide comprehensive manual controls including curves, levels, color balance, hue/saturation, selective color, and channel-specific editing for photographers who want precise control. The adjustment tools support non-destructive editing with adjustment layers that can be modified or removed at any time. RAW processing handles files from every major camera with the quality that photographers expect.

Batch editing applies adjustments, presets, and ML enhancements to multiple photos simultaneously, handling the post-shoot processing workflow where dozens or hundreds of photos need consistent treatment. Copy-paste adjustments apply one photo's corrections to the rest of the set. The batch workflow handles the volume processing that event and production photography generates.

For photographers who want professional-quality photo enhancement with the speed of AI automation and the depth of manual control when needed, Photomator Advanced is the photo editor that makes every photo look its best with minimal effort.

## Performance

Photomator balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
